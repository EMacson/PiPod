# PiPod 🎵

This project recreates the experience of a classic iPod using modern, affordable hardware like a Raspberry Pi, with support for local playback and/or Bluetooth audio streaming.

---

## 🚀 Features

- 🎧 Play music from local storage (`/Music` directory or similar)
- 📡 Optional Bluetooth audio streaming support  
- ⚙️ Lightweight and customizable setup
- 🔁 Auto-connect and startup services

---

## 🧱 Tech Stack

- **Hardware:** Raspberry Pi Pico W
- **Languages:** Python, C
